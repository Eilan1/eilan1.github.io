# Habits Tracker v3.0

A minimal habit tracking application with daily reset, streak tracking, reminders, and achievements.

## Features

- Daily Habit Tracking - Swipe to mark done or missed
- Streak System - Track your consistency over time
- Achievements - Unlock badges for milestones
- Notes and History - Add notes to your completions
- Reminders - Set time-based notifications
- Categories - Organize by Health, Work, Learning, and more
- Difficulty Levels - Easy, Medium, Hard habits
- Themes - Dark mode and custom accent colors
- Analytics - Track success rates and performance
- Data Export - Download JSON backups
- Offline Support - Works without internet, saves locally

## Quick Start

1. Open habitstracker-v3.html in your browser
2. Add your first habit
3. Swipe right to mark done, left for missed
4. Check your stats and achievements

## How to Use

### Adding a Habit
- Click Add button
- Enter habit name
- Select category and difficulty
- Add optional notes
- Confirm

### Tracking
- Swipe Right = Mark as done
- Swipe Left = Mark as missed
- Click Bell = Set reminders
- Click Notebook = Add notes

### Features
- Categories - Filter habits by type
- Reminders - Get notified at custom intervals
- Achievements - Unlock badges as you progress
- Analytics - View detailed performance stats
- Export Data - Backup to JSON file

## Customization

### Themes
Go to Settings to:
- Switch between Dark and Light mode
- Choose accent color
- Enable or disable sound effects
- Toggle delete confirmation

## Data Storage

- All data saved locally in browser storage
- No account needed
- No cloud required
- Your data stays private
- Use Settings > Export to backup as JSON

## FAQ

Q: Will my data be deleted?
A: No. Data saves to browser storage and persists between sessions.

Q: Can I backup my data?
A: Yes. Settings > Export Data downloads a JSON file.

Q: Does it work offline?
A: Yes. Everything works without internet.

Q: Can I use it on mobile?
A: Yes. Fully responsive design.

Q: Is my data shared?
A: No. Everything is stored locally. No tracking, no cloud, no servers.
